AetherX Privacy Policy

Last Updated: October 27th, 2025

As the AetherX development team, we are committed to transparency regarding the data we collect and how it is used. Below is an overview of our data practices:

1. Data Collection

The AetherX Discord Bot collects the following information:

  • User ID + Server:
    • Level
    • XP
    • Permission to Direct Message for suggestions.
  • User ID:
    • Bio
    • Favorite_Color
    • Affiliation
    • AllowStatusView: Allows a user to see the activities of the user via the $profile command
    • DefAllLongOrShort: Toggles the $profile command to show detailed or minimal information
    • Dabloons: Currency for the bot.
    • Inventory: Stores what the user has in their inventory.
    • tgcinv: Stores what cards the user has in their inventory
  • Server Information:
    • Server Name
    • Server ID
    • Server Owner
    • Server Settings (e.g., region, permissions)
  • Website Usage Information:
    • Public Device information
    • Any cookies you give consent to; cookies from the third party; Microsoft Clarity, please see what cookies they use on their website.
    • Information about if you gave consent to cookies, this is required

All other data is retrieved from Discord's public information, and is used strictly for the Bot's functionality.

2. Data Security

The data we collect is used solely to enhance the user experience and improve the bot's functionality. This data is stored securely and only accessed as needed for bot management.

3. No Data Sharing

Since the AetherX Discord bot does not share or sell any user data to third parties, all collected data is kept within the Bot's internal configuration and is used solely for Bot functionality.

4. Data Retention

When the AetherX Discord bot is removed from a server, the data associated with that server is not automatically deleted. This data will remain in the system until manually removed by a bot developer. If you wish to have your data deleted, please contact a bot developer directly to request the removal of your data.

5. Third Party Usage

We use Microsoft Clarity to help improve the user experience on certain parts of our platform. Clarity is not active on the Terms of Service or Privacy Policy pages, and it does not place any cookies until you have given consent. For details on how Clarity handles data, see their Privacy Policy.

The cookie consent box on our homepage is powered by CookieYes, which manages consent for Clarity and other analytics tools. For more information, refer to CookieYes’s Privacy Policy.

6. Requesting User Data

AetherX Bot allows user data to be requested by anyone who provides a valid user ID. This is by design, as the system currently does not implement user authentication or identity verification.

What this means:

  • Anyone can request the data of any user, provided they know the user’s ID.
  • This includes information such as bios, inventory items, preferences, and other stored data as outlined in Section 1.
  • All data retrieved is limited to what is stored in our database and used for the bot's functionality.

Important limitations and considerations:

  • If a user ID is not present in the database, no data will be returned.
  • We do not store data pulled directly from Discord or any other service — only what the bot has saved locally.
  • Frequent or automated requests may be monitored or rate-limited to protect system stability.
  • Due to limitations, we cannot email your data to you.

Users who are concerned about their data visibility may contact a bot developer to request deletion of their stored data or request features that limit public access in future updates. Do note that once someone runs $profile on you, information will be logged again, but only the id and username

You can request your data at https://aether-x.org/PrivPOL/Request-Data

7. Changes to this Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in functionality or compliance requirements. Any updates will be communicated via appropriate channels, such as the Bot or its support server.
